# PakkiOra Frontend

React + Vite app for the PakkiOra appointment booking API.

## Setup

```bash
cd frontend
npm install
copy .env.example .env
npm run dev
```

Open http://localhost:5173

Make sure the backend is running on http://127.0.0.1:8000

## Features

**Patient:** register, login, browse doctors, book/cancel/reschedule appointments, profile

**Admin:** dashboard stats, manage doctors & slots, appointments, patients
